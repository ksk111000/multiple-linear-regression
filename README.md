# Multiple Linear Regression Model

## 📌 Project Overview

This project is a **Multiple Linear Regression (MLR) Machine Learning Model** developed using Python and Scikit-learn. The model predicts output values based on multiple independent variables using regression analysis techniques.

The project demonstrates:

* Data preprocessing
* Feature selection
* Model training and testing
* Prediction using Machine Learning
* Model saving and loading using Pickle (`.pkl`)

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Pickle

---

## 📂 Project Files

* `MLR_Model.pkl` → Trained Multiple Linear Regression model
* `dataset.csv` → Dataset used for training
* `train.py` → Script to train the model
* `predict.py` → Script for predictions
* `README.md` → Project documentation

---

## 📊 Model Workflow

1. Import the dataset
2. Perform data preprocessing
3. Handle missing values and feature selection
4. Split data into training and testing sets
5. Train the Multiple Linear Regression model
6. Evaluate model performance
7. Save the trained model using Pickle
8. Load the model for future predictions

---

## ▶️ Installation & Setup

### Install Required Libraries

```bash
pip install pandas numpy scikit-learn matplotlib seaborn
```

### Run the Training Script

```bash
python train.py
```

### Run Prediction Script

```bash
python predict.py
```
